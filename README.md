# ETSY SCRAPER

Parses products from Etsy.com to JSON. Built in node.js.

## Usage

`npm install`
`npm start`

## Live Demo

https://data.bybumi.com/data

## Known Bugs

Products are parsed without consistency. Different products return different data. I am open to pull requests.
